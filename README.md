# A/H Stock Monitor

轻量级 PyCharm/IntelliJ 插件，用于在 IDE 右侧 ToolWindow 实时查看 A 股、A 股 ETF、港股行情，并维护自选、持仓和当日交易。

## 已实现功能

- 自选分组和持仓分组。
- 支持修改自选/持仓分组名称和持仓股票名称。
- 支持修改持仓数量、可用数量和成本价。
- 左侧分组列表、右键菜单、顶部搜索框和更多菜单。
- 自选列表支持紧凑模式，持仓列表支持底部汇总。
- 支持隐私模式隐藏金额、保存表格列宽和显示刷新时间。
- 支持指数展示，默认包含上证指数、深证成指和创业板指。
- 刷新使用并发请求，提升多股票列表的刷新速度。
- 通过股票代码或名称搜索股票。
- 支持 A 股、A 股 ETF、港股代码归一化。
- 自选列表显示：股票代码、股票名称、当前价格、当日涨幅%、成交额。
- 持仓列表显示：名称/市值、持仓盈亏/比例、持仓/可用、成本/现价、今日收益。
- 支持录入当日买入/卖出，并按 0 轴价格和当日交易记录计算今日收益。
- 今日收益按日内仓位批次计算：当日未交易时使用 `当前持仓 * (现价 - 0轴价)`；有交易时，开盘已有持仓以 0 轴价为基准，当日买入以买入价为基准，卖出按对应批次结算，剩余持仓按实时价浮动。
- 交易后自动维护券商风格动态成本，卖出盈亏会摊入剩余持仓成本，清仓后保留该标的累计持仓盈亏。
- 支持按特定文本格式批量导入。
- 设置页支持整体文本颜色和刷新间隔。

## 导入格式

每行一条，支持 `|`、`,`、制表符或多个空格分隔。

```text
自选|科技|sh600519|贵州茅台
WATCH|ETF|510300|沪深300ETF
持仓|长线|00700|腾讯控股|100|100|320.50
HOLDING|默认|600519|贵州茅台|100|80|1500.00
```

如果只写 `代码 名称`，会导入到当前自选分组：

```text
600519 贵州茅台
00700 腾讯控股
510300 沪深300ETF
```

## 运行

安装 JDK 17 和 Gradle 后执行：

```shell
gradle runIde
```

或直接用 PyCharm/IntelliJ IDEA 打开本目录，导入 Gradle 项目后运行 `runIde`。
